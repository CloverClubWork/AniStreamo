# AniStreamo - Stream Anime for Free
~ Is an application that provides free streaming anime from various site's without ad interruptions.

This version is on beta mode, which meant it is not yet complete and is still in testing mode.

### Why use AniStreamo?
- Ad Interruptions Blocked (This application did not blocked ads including Popup Ad as it is but instead prevented them from sending you to another site which would stop your streaming and prove to be irritating)

### Server supported
- [9Anime](https://9anime.to/home)
- [Animepahe](https://animepahe.com/)
- [Anihdplay](https://anihdplay.com/)
